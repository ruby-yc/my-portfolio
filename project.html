AI mask detect system

Developed a web-based application capable of detecting whether individuals are wearing face masks using computer vision techniques. The system utilizes a real-time object detection model integrated into a web interface to provide instant sound feedback. In addition to the software component, I independently designed and connected the hardware setup using an Arduino microcontroller, incorporating sensors and buzzer for data input and control.
